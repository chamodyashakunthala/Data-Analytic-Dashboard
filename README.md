# AnalyticOS — Business Analytics Dashboard

A full-featured business analytics dashboard built with React, Recharts, and Tailwind CSS.

## Features
- 4 KPI cards with sparkline charts and trend indicators
- Revenue overview with Area/Bar chart toggle
- Product performance bars and traffic source donut chart
- Sortable regional breakdown table with drill-down
- Live real-time activity feed (toggleable)
- AI-powered strategic recommendations
- Dark theme with animated UI

## Tech Stack
- React 18
- Recharts
- Tailwind CSS
- Vite

## Getting Started
```bash
npm install
npm run dev
```

Open http://localhost:5173
